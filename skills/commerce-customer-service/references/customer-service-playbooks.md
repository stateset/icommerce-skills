# Customer Service Playbooks

## Where is my order?

1. Find customer by email
2. Get latest order
3. Share status and tracking

## I want a return

1. Confirm order and items
2. Create return request
3. Approve/reject based on policy
4. Issue refund if approved

## Item out of stock

1. Check inventory for SKU
2. Suggest alternative or backorder
3. Notify when restocked

## Cancel order

1. Verify order status
2. Cancel if not shipped
3. Otherwise convert to return
